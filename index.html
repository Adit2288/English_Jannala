FAQ

Common JavaScript interview questions.

Difference between var, let, and const

var is function-scoped; let and const are block-scoped. const can’t be reassigned.

Difference between map(), forEach(), and filter()

map() returns a new array, forEach() returns nothing, filter() returns matching items.

Arrow functions vs regular functions

Arrow functions don’t bind their own this; regular functions do.

How JavaScript Promises work

Promises represent async operations with states: pending → fulfilled/rejected. Use then/catch/finally.

How closures work in JavaScript

A closure lets a function remember variables from its outer scope even after it returns.
